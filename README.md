# 🎶 Python Music Player

Welcome to **PythonGeeks Music Player**! 🔥

## Features ✨

- 🎵 Play Music
- ⏸️ Pause Music
- ▶️ Resume Music
- ⏹️ Stop Music
- 📂 Load Songs from Folder
- 📜 Playlist View with Scrollbar

## Technologies Used 🛠️

- Python
- Tkinter (GUI Framework)
- Pygame Mixer (Audio Library)

## How to Run 🚀

```bash
pip install tkinter
pip install pygame
python music player.py
```
